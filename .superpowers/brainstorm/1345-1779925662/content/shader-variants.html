GalacticShaderProvider

Extends libGDX DefaultShaderProvider. For each renderable, reads MaterialVariantComponent from the entity and selects/caches a shader keyed by its feature flags (HAS_FACTION_TINT | HAS_DAMAGE | HAS_EMISSIVE). Shader programs are compiled once and pooled — a character without emissive never pays for that shader path.

MaterialVariantComponent

Ashley component holding: Vector3 factionColor int damageState 0-3 float emissiveScale. Written each frame by FactionSystem, HealthSystem, and PowerSystem respectively via event bus. The shader reads these as uniforms — no model reload needed to change faction or take damage.
For the stylized art style, damage overlays are color-shift + emissive cracks — not PBR normal maps or displacement. This means the same texture set works for all damage states; the shader does all the work via a u_damageState uniform that controls desaturation amount, crack mask sampling, and crack emissive color. Much cheaper than separate texture variants per tier.
