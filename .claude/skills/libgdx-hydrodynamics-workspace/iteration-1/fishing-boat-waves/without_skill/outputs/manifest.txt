Fishing Boat Water Physics - File Manifest
===========================================

Files created for the fishing boat ECS implementation:

1. WaveQuery.java
   Stateless utility for evaluating Gerstner wave surface height and normals at galaxy-space coordinates.
   Package: com.galacticodyssey.water

2. BoatBuoyancySystem.java
   Per-point buoyancy system that transforms hull sample points to world space, queries wave height via WaveSystem using galaxy-space coordinates, and applies forces along hull normals through btRigidBody.applyForce for natural restoring torques.
   Package: com.galacticodyssey.water.systems

3. HydrodynamicDragSystem.java
   Axis-decomposed quadratic drag system (skin friction + form drag + wave-making drag) that subtracts water current velocity before computing drag, with quadratic angular damping for heavy turning feel. All forces via btRigidBody.applyCentralForce/applyTorque.
   Package: com.galacticodyssey.water.systems

4. BoatMotorSystem.java
   Translates BoatInputComponent throttle/steering into thrust and rudder forces applied through Bullet physics. Uses exponential smoothing for weighty analog feel. Rudder effectiveness scales with forward speed.
   Package: com.galacticodyssey.water.systems

5. FishingBoatFactory.java
   Assembles a complete fishing boat entity with 14 buoyancy sample points distributed across bow, stern, port, starboard, keel, and chines. Creates the Bullet rigid body, HullComponent, BoatMotorComponent, and BoatInputComponent.
   Package: com.galacticodyssey.water

Pre-existing files used (not created by this task):
- WaterBodyComponent.java (water/): Wave params, density, current velocity
- WaveParams.java (water/): Single Gerstner wave definition
- BuoyancySamplePoint.java (water/): Hull sample point with local offset, normal, area
- HullComponent.java (water/): Hull geometry, drag coefficients, sample points array
- WakeComponent.java (water/): Froude number and wake trail for rendering
- BoatInputComponent.java (water/components/): Throttle and steering input
- BoatMotorComponent.java (water/components/): Motor thrust and rudder parameters
- WaveSystem.java (water/systems/): Service system for wave height/normal/flow queries
